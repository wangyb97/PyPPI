Installation
================

Prerequisites
~~~~~~~~~~~~~~~~
The following packages are requirements:

- ``gensim``
- ``glove``
- ``glove_python_binary``
- ``Keras``
- ``matplotlib``
- ``numpy``
- ``scikit_learn``
- ``seaborn``
- ``shap``
- ``tensorflow`` or ``tensorflow_gpu``
- ``torch``
- ``transformers``
- ``yellowbrick``
